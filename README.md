# GameOfLife

This project is a Python-based simulation inspired by Conway's Game of Life. The program models a simple ecosystem in a grid-based "petri dish" where cells grow, reproduce, and die based on local population rules.

In addition to the classic Game of Life mechanics, this simulation introduces predators that move randomly through the environment and consume living cells, creating emergent population dynamics similar to simple biological systems.

The simulation runs in the terminal using ASCII graphics and allows the user to manually place cells before starting the simulation.

Features
Classic Game of Life rules
Predator organisms that hunt cells
Interactive grid editing with keyboard controls
Real-time terminal rendering
Restartable simulation environment

Controls
W / A / S / D or Arrow Keys – Move cursor
Space – Place a living cell
B – Begin simulation
P – Pause simulation
R – Reset the world

Technologies
Python
Terminal ASCII rendering
Keyboard input handling

Goal
This project explores how complex behaviors and patterns can emerge from simple rules, similar to biological processes observed in microbial colonies.
